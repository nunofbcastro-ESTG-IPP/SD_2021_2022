<h2>How to run</h2>

<h3>Database</h3>

```
cd Database
docker compose up
```

<h3>Server</h3>

```
cd sd_grupo6_server
gradle run
```

<h3>Client</h3>

```
cd sd_grupo6_client
gradle run
```
